  src/app/layout.tsx                   ← 全局 Root Layout (服务端)

  ---
  📦 Provider / Store 架构

  PublicConfigProvider          ← 从 /api/v1/public-config 拉取站点配置
  └── AuthProvider              ← 从 localStorage 恢复登录态
      └── pages

  usePublicConfig() hook        ← 站点名称、验证码开关、注册开关、备案信息...
  useAuthStore()                ← user, isAuthenticated, isAdmin, login(),
  logout()

  📋 总结关键数据

  - 总页面数：28 个（公共 8 + 用户 5 + 管理 15）
  - Layout 层级：Root → (PublicConfig + Auth) → 各页
  - 现有 Shell 组件：SiteShell（公共）、DashboardShell（用户，简陋）、AdminShell
  （管理，完善的三栏）
  - 缺少的路由（Dashboard 侧边栏需求）：/dashboard/usage、/dashboard/logs
