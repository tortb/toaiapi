## E2E 测试提示词

```
为 ToAIAPI 编写 E2E 测试。

测试流程：
{流程描述}

要求：
1. 模拟真实用户操作
2. 测试完整业务流程
3. 包含认证流程
4. 验证数据库状态
5. 验证响应格式

关键流程必须覆盖：
- 用户注册 → 登录 → 创建 API Key → 调用 Gateway → 查看余额
- 管理员登录 → 创建 Provider → 创建 Channel → 创建 Model → 设置定价
```
