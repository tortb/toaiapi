# ToAIAPI — 产品愿景

## 一句话定义

**ToAIAPI 是一个企业级 AI API 网关平台，统一管理所有主流 AI 模型的接入、计费和调用。**

## 目标用户

| 用户类型 | 场景 | 核心诉求 |
|---------|------|---------|
| 个人开发者 | 日常 AI 开发 | 一个 Key 调用所有模型，按量计费 |
| 技术团队 | 产品集成 | 统一 API、用量统计、成本控制 |
| 企业客户 | 大规模调用 | SLA 保障、私有部署、财务对账 |
| 代理商 | 分销 AI 能力 | 多级定价、客户管理、利润报表 |

## 核心价值

1. **统一接入** — OpenAI 兼容格式，一个 SDK 调用 OpenAI / Claude / Gemini / DeepSeek 等所有模型
2. **智能路由** — 自动选择最优渠道（成本、延迟、成功率），故障自动转移
3. **精确计费** — Token 级别精确计费，支持输入/输出/缓存/推理分别计价
4. **企业就绪** — 多租户、RBAC、审计日志、财务对账

## 商业模式

- **按量计费** — 按 Token 消耗收费，加价率由平台控制（multiplier）
- **订阅制** — 月付/年付套餐，含固定配额
- **代理商** — 多级分销，代理商自定义定价
- **企业版** — 私有部署 + 定制化，按年付费

## 技术愿景

- 所有金额以 **分（fen）** 为单位，杜绝浮点精度问题
- 所有 API Key 使用 **Argon2id** 哈希存储
- 所有敏感操作使用 **数据库事务** 保证一致性
- Gateway 层 **不信任** Provider 返回的 Token 数，自行计算
