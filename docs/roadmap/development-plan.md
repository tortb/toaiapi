# ToAIAPI Development Roadmap

## V1.0 - 核心基础（MVP）

**目标**: 跑通基本流程，验证商业模式

### 功能清单

#### 用户系统
- [ ] 邮箱注册/登录
- [ ] JWT 认证
- [ ] 用户信息管理
- [ ] 密码重置

#### API Key 系统
- [ ] 创建 API Key
- [ ] 删除 API Key
- [ ] API Key 验证
- [ ] 基本限流

#### 网关核心
- [ ] OpenAI 兼容接口 (`/v1/chat/completions`)
- [ ] 基本请求转发
- [ ] 流式响应支持
- [ ] 错误处理

#### 计费系统
- [ ] Token 使用记录
- [ ] 基本费用计算
- [ ] 余额扣减
- [ ] 请求日志

#### 余额系统
- [ ] 余额查询
- [ ] 基本充值（手动）
- [ ] 交易流水

### 技术任务

- [ ] Monorepo 搭建
- [ ] NestJS 项目初始化
- [ ] Prisma Schema 设计
- [ ] PostgreSQL 配置
- [ ] Redis 配置
- [ ] 基本 CI/CD

### 里程碑

- [ ] 用户可以注册登录
- [ ] 用户可以创建 API Key
- [ ] 用户可以通过 API Key 调用 OpenAI 接口
- [ ] 系统可以记录 token 使用并扣费
