# ToAIAPI 开发路线图

## V1.0 - 核心基础（MVP）

**目标**: 跑通基本流程，验证商业模式

**状态**: ✅ 已完成

### 用户系统

- [x] 邮箱注册 / 登录
- [x] JWT 认证（Access Token + Refresh Token）
- [x] 用户信息管理
- [x] 密码重置（忘记密码 + 修改密码）
- [x] 登出（Token 撤销）

### API Key 系统

- [x] 创建 API Key
- [x] 删除 API Key
- [x] API Key 验证（Guard）
- [x] 启用 / 禁用 API Key
- [x] 更新 API Key 配置
- [x] 基本限流（多级：短期/中期/长期）
- [x] 模型白名单
- [x] IP 白名单

### 网关核心

- [x] OpenAI 兼容接口（`/v1/chat/completions`）
- [x] 基本请求转发
- [x] 流式响应支持（SSE）
- [x] 错误处理与异常过滤
- [x] 模型列表接口（`/v1/models`）
- [x] 请求限流（ThrottlerGuard）

### 计费系统

- [x] Token 使用记录
- [x] 基本费用计算（`@toai/billing` 包）
- [x] 余额扣减（数据库事务）
- [x] 请求日志（RequestLog）
- [x] 模型定价管理

### 余额系统

- [x] 余额查询
- [x] 基本充值（管理员手动）
- [x] 交易流水查询

### 技术任务

- [x] Monorepo 搭建（pnpm workspace + Turbo）
- [x] NestJS 项目初始化
- [x] Prisma Schema 设计
- [x] PostgreSQL 配置
- [x] Redis 配置
- [x] 前端页面（登录、注册、密码重置、仪表盘、API Key 管理、使用记录、设置）

### 里程碑

- [x] 用户可以注册登录
- [x] 用户可以创建 API Key
- [x] 用户可以通过 API Key 调用 OpenAI 兼容接口
- [x] 系统可以记录 token 使用并扣费
- [x] 前端仪表盘展示余额和请求日志
